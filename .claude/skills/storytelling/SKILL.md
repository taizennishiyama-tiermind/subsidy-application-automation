# 申請書ストーリー構成スキル

## トリガー

`/draft` コマンド実行時に自動で参照される。
採択される申請書に共通する「物語の構造」を使って、説得力のある申請書を構成する。

---

## 基本原則

補助金の審査員は大量の申請書を短時間で読む。採択される申請書には共通する「物語の構造」がある。
審査員が「この会社を応援したい」「この投資は合理的だ」と感じる申請書を作る。

## 基本構造: 4部構成

```
1. 現状（Where we are）
   └ 自社の強み・実績を示す

2. 課題（What's wrong）
   └ 環境変化・顧客ニーズ・内部課題を示す

3. 解決策（How we fix it）
   └ 補助事業の具体的な取組を示す

4. 期待効果（Where we'll be）
   └ 数値目標・波及効果を示す
```

---

## パターン1: 「強みを活かした革新」型

**適合する補助金**: ものづくり補助金

```
[現状] 当社は○○業界で○年の実績があり、△△技術に強みを持つ。
       顧客から高い評価を得ており、リピート率は○○%。

[課題] しかし近年、市場環境の変化（具体例）により、
       従来の方法では対応が困難になってきている。
       具体的には、①○○ ②△△ ③□□ という課題がある。

[解決策] そこで本事業では、○○設備を導入し、
         革新的な△△プロセスを構築する。
         これにより、従来不可能だった□□が実現可能となる。

[効果] 結果として、生産性が○%向上し、
       新規顧客○社の獲得が見込まれる。
       付加価値額は年平均○%以上の成長を達成する。
```

## パターン2: 「危機からの脱却」型

**適合する補助金**: 新事業進出補助金

```
[現状] 当社の主力事業は○○であり、年商○億円。
       ○○を強みとしてきた。

[課題] しかし、○○（外部環境の変化）により、
       主力事業の売上が○年連続で減少。
       このままでは○年後に○○という深刻な事態が想定される。

[解決策] そこで当社の○○という強みを活かし、
         新たに△△分野に進出する。
         具体的には○○設備を導入し、□□サービスを開始する。

[効果] 新規事業により○年後に売上○万円を見込む。
       既存事業とのシナジーにより、会社全体の売上は○%回復。
       従業員の雇用も維持・拡大する。
```

## パターン3: 「DX・デジタル化」型

**適合する補助金**: IT導入補助金 / デジタル化・AI導入補助金

```
[現状] 当社は○○業を営み、従業員○名で運営。
       ○○業務は手作業・紙ベースで行っている。

[課題] ①業務の属人化（特定社員にしかできない）
       ②処理時間がかかる（月○時間）
       ③ヒューマンエラーが発生（年○件）
       ④データ活用ができていない

[解決策] ○○システム（ITツール名）を導入し、
         △△業務を自動化・デジタル化する。
         これにより○○が実現する。

[効果] 作業時間が月○時間から△時間に削減（○%削減）。
       エラー率が○%から△%に低減。
       空いたリソースを顧客対応に振り向け、売上○%向上。
```

## パターン4: 「販路拡大」型

**適合する補助金**: 小規模事業者持続化補助金

```

       地域の○○層を主な顧客としている。

         △△市場への販路を開拓する。

```

## ストーリー構成のテクニック

### 1. 「具体性」で説得力を出す

```
× 「売上が減少している」
○ 「直近3年間で売上が15%減少し、2023年度は前年比92%の8,500万円」

× 「生産性が向上する」
○ 「1ロットあたりの生産時間が120分から45分に短縮（62.5%削減）」
```

### 2. 「外部データ」で裏付ける

- 業界の市場規模データ（○○白書、業界団体統計等）
- 地域の人口動態データ（総務省統計等）
- 顧客アンケート結果
- 競合他社の動向

### 3. 「before/after」を明確にする

| 項目 | 現状（before） | 導入後（after） | 改善率 |
|------|-------------|--------------|--------|
| 生産量 | 100個/日 | 200個/日 | 100%UP |
| 不良率 | 5% | 1% | 80%DOWN |
| 納期 | 2週間 | 3日 | 78%短縮 |

### 4. 「審査基準」に直接対応する

公募要領の審査項目を一つずつチェックし、申請書内で全て言及されているか確認する。
審査項目に対応する記述がない = 減点される。

### 5. 図表・画像を効果的に使う

- 事業フロー図（現状 vs 導入後）
- 市場データのグラフ
- 製品・設備の写真
- 組織体制図
- スケジュール表（ガントチャート）

## 「逆算型」ストーリーの組み立て方

事業者が「○○設備を買いたい」と言ってきた場合の逆算手順:

```
Step 1: 設備の特定 → 「何を買いたいのか？」
Step 2: 効果の逆算 → 「その設備で何ができるようになるのか？」
Step 3: 課題の逆算 → 「なぜ今、それが必要なのか？」
Step 4: 現状の整理 → 「自社にはどんな強みがあるのか？」
Step 5: ストーリーの統合 → 現状の強み → 環境変化による課題 → 設備投資による解決 → 数値的効果
```

## 文章表現のポイント

### やるべきこと
- 一文一義（1つの文に1つの情報）
- 主語と述語を近くに置く
- 箇条書きで整理する
- 専門用語には簡単な説明を添える

### やってはいけないこと
- 抽象的な表現だけで終わる
- 根拠のない楽観的な数値
- 補助金依存の計画（補助金がなくても事業継続できることを示す）
- コピペ感のある文章（テンプレートの丸写し）
