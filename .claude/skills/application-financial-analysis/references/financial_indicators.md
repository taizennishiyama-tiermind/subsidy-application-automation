# 財務指標リファレンス

## 収益性

- 営業利益率 = 営業利益 / 売上高
- ROA = 当期純利益 / 総資産
- ROE = 当期純利益 / 純資産

## 安全性

- 自己資本比率 = 純資産 / 総資産
- 流動比率 = 流動資産 / 流動負債
- 固定比率 = 固定資産 / 純資産

## 成長性

- 売上高成長率 = (当期売上高 - 前期売上高) / 前期売上高
- 営業利益成長率 = (当期営業利益 - 前期営業利益) / 前期営業利益

## 返済能力

- 債務償還年数 = 有利子負債 / (営業利益 + 減価償却費)
- インタレストカバレッジレシオ = 営業利益 / 支払利息

## 補助金申請で重視する追加指標

- 付加価値額 = 営業利益 + 人件費 + 減価償却費
- 給与支給総額
- 労働生産性 = 付加価値額 / 従業員数
- 最低賃金対比

## 読み方の目安

- 自己資本比率 30%以上: おおむね良好
- 流動比率 120%以上: 短期支払余力あり
- 営業利益率 5%以上: 本業収益力あり
- 債務償還年数 10年以下: おおむね許容

数値の良し悪しだけでなく、申請書では「なぜそうなっているか」と「補助事業でどう改善するか」を添える。
