# 実現性チェックリスト

順位付け前に次を確認する。

## 期限

- [ ] 申請締切日を確認した
- [ ] 残り日数を確認した
- [ ] 締切までにヒアリング、ドラフト、レビュー、テンプレート反映まで間に合う

## 要件

- [ ] 企業規模要件を満たす
- [ ] 業種要件を満たす
- [ ] 地域要件を満たす
- [ ] 投資内容が対象経費に入る
- [ ] 数値要件を満たせる見込みがある

## 事前準備

- [ ] gBizID 等の事前IDが間に合う
- [ ] 決算書、見積書、会社資料を揃えられる
- [ ] 加点資料がある、または取得可能

## 申請戦略

- [ ] 審査基準に対する強みが明確
- [ ] 弱点に対する補完策がある
- [ ] 申請書ストーリーを組み立てやすい
- [ ] 第一候補として無理がない

## 判定の目安

- 3 つ以上の重大未確定がある候補は、第一候補から外す
- 締切 1 週間以内は原則として「相当急ぎ」の注記を付ける
