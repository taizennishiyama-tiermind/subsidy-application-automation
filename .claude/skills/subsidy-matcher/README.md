# 補助金マッチングスキル

## 概要

案件情報と制度要件を照合し、現実的に進められる補助金候補を絞るためのスキルです。
この skill は、単なる制度検索ではなく、次の 3 つを同時に見ることを目的にしています。

- 制度上の適合
- 準備負荷と締切
- 申請書としての訴求しやすさ

## 主な機能

- `grants/` に蓄積した補助金情報の横断比較
- 追加調査が必要な場合の検索方針整理
- 6軸スコアリングによる候補比較
- 準備負荷を加味した実現性チェック
- `strategy.md` に落としやすい形式での結果整理

## 出力イメージ

- 第一候補から第三候補
- 補助額と補助率
- 通りやすい訴求ポイント
- 締切と準備負荷
- 逆転条件
- 次にやること

## 実務上のポイント

- 制度相性が高くても締切が近すぎる場合は第一候補にしない
- 地域補助金は「自治体公式サイトで募集中か」を必ず確認する
- 補助率だけでなく、自己負担額と準備負荷を見る
- 採択率が不明な場合は、審査基準との噛み合いと準備負荷から実質的な難易度を推定する
