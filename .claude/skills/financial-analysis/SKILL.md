# 財務分析スキル

## トリガー

`/analyze` コマンド実行時に自動で参照される。
決算書等の財務データを分析し、補助金申請の数値計画に説得力を持たせる。

---

## 入力データの種類

### 必須データ
- 直近3期分の決算書（貸借対照表・損益計算書）
- 法人事業概況説明書
- 確定申告書

### あると望ましいデータ
- 月次試算表（直近分）
- 資金繰り表
- 設備投資計画
- 借入金一覧

### 個人事業主の場合
- 確定申告書（青色申告決算書）
- 収支内訳書

---

## 分析の4つの観点

### 1. 収益性分析

| 指標 | 計算式 | 補助金申請での活用 |
|------|--------|------------------|
| 売上高 | - | 推移を見て成長性・課題を把握 |
| 営業利益率 | 営業利益 / 売上高 | 本業の収益力を示す |
| 経常利益率 | 経常利益 / 売上高 | 全体の収益力 |
| 付加価値額 | 営業利益 + 人件費 + 減価償却費 | **補助金の主要要件** |
| 労働生産性 | 付加価値額 / 従業員数 | IT導入補助金等で重要 |

#### 付加価値額の算出（最重要）

```
付加価値額 = 営業利益 + 人件費 + 減価償却費

※ ものづくり補助金: 年平均 +3% 以上の成長が必須要件
※ 新事業進出補助金: 年平均 +3〜4% 以上の成長が必須要件
```

### 2. 安全性分析

| 指標 | 計算式 | 補助金申請での活用 |
|------|--------|------------------|
| 自己資本比率 | 自己資本 / 総資産 | 財務の健全性を示す |
| 流動比率 | 流動資産 / 流動負債 | 短期的な支払能力 |
| 借入金月商倍率 | 借入金 / 月商 | 借入の適正度 |

### 3. 成長性分析

| 指標 | 計算式 | 補助金申請での活用 |
|------|--------|------------------|
| 売上高成長率 | (当期売上 - 前期売上) / 前期売上 | 事業の成長トレンド |
| 経常利益成長率 | 同上（利益ベース） | 収益の成長トレンド |
| 設備投資額推移 | - | 投資意欲・成長意欲 |

### 4. 人件費分析

| 指標 | 計算式 | 補助金申請での活用 |
|------|--------|------------------|
| 給与支給総額 | - | **ものづくり補助金の要件** |
| 一人当たり人件費 | 人件費 / 従業員数 | 賃上げ計画の根拠 |
| 労働分配率 | 人件費 / 付加価値額 | 賃上げ余力の判断 |
| 最低賃金対比 | 実際の最低時給 / 都道府県最低賃金 | 賃上げ要件の確認 |

---

## 補助金別の重点分析項目

### ものづくり補助金
- **付加価値額**: 年平均 +3% 以上の計画が必須
- **給与支給総額**: 年平均 +2% 以上の計画が必須
- **最低賃金**: 都道府県別 +30円以上
- 設備投資の費用対効果（ROI）

### IT導入補助金
- **労働生産性**: 導入前後の比較
- 作業時間・コストの定量化
- 売上高・経常利益の改善見込み

### 新事業進出補助金
- **付加価値額**: 年平均 +3〜4% 以上
- 新事業の収益計画の妥当性
- 既存事業とのバランス

### 小規模事業者持続化補助金
- 売上高の推移（課題の根拠）
- 投資効果の見込み
- 費用の妥当性

## 注意点

### やってはいけないこと
- 根拠のない右肩上がりの数値計画
- 決算書との矛盾
- 付加価値額の算出間違い
- 人件費を含めない付加価値額計算

### よくある間違い
- 付加価値額に「人件費」を含め忘れる
- 減価償却費の計上タイミングを間違える
- 従業員数の定義（パート含むか等）の不一致
- 売上と利益の整合性が取れていない

## PDFデータの読み取り

Claude Code は PDF ファイルを直接読み取ることができる。
事業者から決算書PDFを受け取った場合:
1. PDFファイルを `workspace/{案件名}/` に配置
2. Claude に読み込みを依頼
3. 上記の分析フレームワークに沿って分析を実施
4. 結果を `workspace/{案件名}/company_data.md` に記録
