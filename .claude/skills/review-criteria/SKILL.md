# 審査基準・レビュースキル

## トリガー

`/review` コマンド実行時に自動で参照される。
`/match` コマンドでも補助金選定の判断材料として参照する。

---

## 審査の全体像

補助金の審査は、外部有識者（中小企業診断士、大学教授、業界専門家等）が行う。
審査員一人あたり数十〜数百件の申請書を読むため、**わかりやすさ**が最も重要。

### 審査員の心理
- 短時間で判断する必要がある → 最初の1ページで印象が決まる
- 全ての審査項目をチェックする → 項目漏れは即減点
- 数値の根拠を確認する → 根拠なき数値は不信感を生む
- 「この会社を応援したい」と思えるか → 情熱と合理性の両立

## 補助金別の審査基準

### ものづくり補助金

#### 技術面（配点高）
| 審査項目 | 評価ポイント | 申請書での対応 |
|---------|------------|-------------|
| 革新性 | 新製品・新サービス・新生産方式として革新的か | 「従来にない」ポイントを明確に記述 |
| 課題の明確性 | 課題が具体的に特定されているか | データ・具体例で課題を示す |
| 解決方法の妥当性 | 技術的に実現可能か | 自社の技術力・経験で裏付け |
| 補助事業の具体性 | スケジュール・実施体制が明確か | ガントチャート・体制図を添付 |

#### 事業化面（配点高）
| 審査項目 | 評価ポイント | 申請書での対応 |
|---------|------------|-------------|
| 市場ニーズ | ターゲット市場が明確か | 市場規模データ・顧客の声 |
| 事業化の見通し | 販売先・販路が具体的か | 既存顧客・引合い情報 |
| 費用対効果 | 投資に見合うリターンか | ROI計算を明示 |
| 数値計画 | 根拠ある計画か | 積み上げ方式で算出 |

#### 政策面
| 審査項目 | 評価ポイント | 申請書での対応 |
|---------|------------|-------------|
| 地域経済貢献 | 地域の雇用・経済に寄与するか | 雇用計画・地域連携 |
| 環境配慮 | 環境負荷低減に寄与するか | 省エネ・脱炭素の取組 |
| ニッチトップ | 特定分野での差別化があるか | 独自技術・特許等 |

### IT導入補助金

| 審査項目 | 評価ポイント | 申請書での対応 |
|---------|------------|-------------|
| 課題認識 | 自社の経営課題を理解しているか | SWOT分析等で整理 |
| ツール適合性 | 課題とITツールがマッチしているか | 機能と課題の対応表 |
| 導入効果 | 具体的な改善効果が見込めるか | 作業時間・コストの定量比較 |
| 継続性 | 導入後も活用し続けられるか | 運用体制・教育計画 |

### 新事業進出補助金

| 審査項目 | 評価ポイント | 申請書での対応 |
|---------|------------|-------------|
| 新事業の必然性 | なぜ新事業に進出する必要があるか | 外部環境分析・危機感 |
| 市場分析 | 新事業の市場は魅力的か | 市場規模・成長性データ |
| シナジー | 既存事業との相乗効果はあるか | 技術・顧客・ノウハウの転用 |
| リスク対策 | リスクを認識し対策しているか | リスク分析表 |

### 小規模事業者持続化補助金

| 審査項目 | 評価ポイント | 申請書での対応 |
|---------|------------|-------------|
| 経営状況分析 | 自社の現状を正しく把握しているか | 具体的な数値で整理 |
| 一貫性 | 経営方針と補助事業の関連性 | 方針→目標→事業計画のつながり |
| 有効性 | 補助事業が販路開拓に有効か | 期待される効果の定量化 |
| 積算の適切性 | 経費が妥当か | 見積書・市場価格との比較 |

## 加点項目の活用（全補助金共通）

加点項目は「取れるものは全て取る」のが鉄則。

### 主な加点項目
1. **賃上げ加点** - 事業所内最低賃金の引上げ計画
2. **経営革新計画** - 都道府県知事の承認を取得
3. **事業継続力強化計画** - 防災・減災の取組
4. **パートナーシップ構築宣言** - 取引先との共存共栄
5. **くるみん認定/えるぼし認定** - 子育て支援・女性活躍
6. **デジタル技術活用** - DXの取組

### 加点取得の優先順位
1. 賃上げ加点（最も加点が大きい傾向）
2. 事業継続力強化計画（申請が比較的容易）
3. パートナーシップ構築宣言（オンラインで即日可能）
4. 経営革新計画（審査に時間がかかるため早めに準備）

## 審査で落ちる典型的パターン

### パターン1: 「設備購入の言い訳」型
- 補助金で設備を買うことが目的化
- → 改善: 設備は手段、目的は事業成長であることを示す

### パターン2: 「夢物語」型
- 根拠のない楽観的な数値計画
- → 改善: 過去データ・市場データからの積み上げ

### パターン3: 「コピペ」型
- テンプレートの丸写し感がある
- → 改善: 自社独自のエピソード・数値を多く入れる

### パターン4: 「項目漏れ」型
- 審査基準の一部に対応する記述がない
- → 改善: 公募要領のチェックリストで全項目カバーを確認

### パターン5: 「読みにくい」型
- 文字だらけで図表がない
- → 改善: 図表を入れ、段落ごとに主張を明確に

## セルフレビューチェックリスト

申請書完成後、以下を確認:

### 形式面
- [ ] ページ数制限を守っているか
- [ ] 必須記載項目を全てカバーしているか
- [ ] 誤字脱字がないか
- [ ] 図表が適切に配置されているか

### 内容面
- [ ] ストーリーに一貫性があるか（現状→課題→解決→効果）
- [ ] 数値に根拠があるか
- [ ] 審査基準の各項目に対応する記述があるか
- [ ] 自社の強みが明確に伝わるか
- [ ] 補助事業の具体的な取組が明確か
- [ ] 実施スケジュールが妥当か

### 要件面
- [ ] 付加価値額の要件を満たす計画になっているか
- [ ] 給与支給総額の要件を満たす計画になっているか
- [ ] 最低賃金の要件を確認したか
- [ ] 加点項目を可能な限り取得しているか
- [ ] 必要な添付書類が全て揃っているか

## 採択事例パターン集（匿名化）

### 事例1: 金属加工業 × ものづくり補助金
- 従業員15名、年商2億円、申請額1,000万円
- 5軸マシニングセンタ導入で複雑形状の一貫加工体制を構築
- **採択ポイント**: 顧客からの引き合いという具体的ニーズ、外注費500万円削減の定量効果、技術継承への対応

### 事例2: 不動産管理業 × IT導入補助金
- 従業員8名、年商1.5億円、申請額150万円
- クラウド型物件管理システム導入
- **採択ポイント**: 家賃集計3日→2時間（94%削減）の具体的before/after、属人化解消

### 事例3: 和菓子製造小売 × 小規模事業者持続化補助金
- 従業員3名、年商2,000万円、申請額50万円
- ECサイト構築 + パッケージリニューアル
- **採択ポイント**: 創業50年の伝統、人口減少という地域課題、事業承継の視点

### 事例4: レストラン × 新事業進出補助金
- 従業員12名、年商8,000万円、申請額2,500万円
- セントラルキッチン新設 + 冷凍食品販売
- **採択ポイント**: 看板メニューを活かしたシナジー、BtoC/BtoB両面展開

### 共通する成功要因
1. **自社の強みが明確**: 「なぜこの会社がやるべきか」が伝わる
2. **課題が具体的**: 数値データで裏付けされた課題認識
3. **解決策との論理的なつながり**: 課題→解決策→効果の因果関係が明確
4. **定量的な効果**: before/afterの数値比較
5. **実現可能性**: 過去の実績や能力で裏付けされた計画
6. **社会性**: 地域経済、雇用、環境への貢献
