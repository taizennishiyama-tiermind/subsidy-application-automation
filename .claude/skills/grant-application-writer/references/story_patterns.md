# ストーリーパターン

## A. 課題解決型

- 明確なボトルネックがある
- 設備や IT 導入で詰まりを解消する
- 効果は時間短縮、能力増強、不良率低減で見せる

## B. 成長機会型

- 需要はあるが供給力が足りない
- 投資で受注拡大や市場シェア拡大を狙う
- 既存顧客の追加受注や引き合いを根拠にする

## C. 変革・転換型

- 既存市場の縮小や外部環境変化がある
- 新分野進出や業態転換が主題
- 既存の強みが新分野にどう転用できるかを中心に書く

## D. 差別化・高付加価値型

- 価格競争から抜けたい
- 精度、品質、納期、独自技術で差別化する
- 単価改善と利益率改善の筋道を示す

## 選び方

- 課題が先に立つなら A
- 需要拡大が先に立つなら B
- 市場転換が必要なら C
- 競争優位の再構築なら D

複合型でもよいが、主軸は1つに決める。
