# 補助金審査基準リファレンス

このファイルは、テンプレート要件が不明な場合に参照する一般的な審査基準をまとめたものです。

---

## 一般的な審査項目（多くの補助金に共通）

### 1. 技術的優位性・革新性（配点目安: 30-40%）

**評価ポイント:**
- 既存技術との明確な差別化
- 技術的実現可能性
- 特許・知財の有無
- 他社との比較（定量的）

**良い申請書の特徴:**
- 「従来比○○%向上」など定量的な比較
- 競合他社製品との性能比較表
- 開発リスクと対策が明記されている
- 技術的根拠（論文、試験データ等）がある

**減点される例:**
- 「画期的な技術です」など抽象的表現のみ
- 他社比較がない
- 実現可能性の根拠がない
- リスク認識が甘い

---

### 2. 事業化実現性（配点目安: 30-40%）

**評価ポイント:**
- 市場分析の具体性
- 売上計画の妥当性
- 実施体制の適切性
- スケジュールの現実性

**良い申請書の特徴:**
- 市場規模のデータソースが明記されている
- 売上計画に複数の積算根拠（市場シェア、顧客ベース等）
- 実施体制に具体的な役割分担と担当者名
- ガントチャートで工程が可視化されている
- 既存顧客や引き合い実績が示されている

**減点される例:**
- 「市場は拡大傾向」など根拠のない記述
- 売上計画の積算根拠なし
- 「社員一同で頑張ります」など抽象的な実施体制
- スケジュールが楽観的すぎる

---

### 3. 政策適合性（配点目安: 10-20%）

**評価ポイント:**
- 補助金の政策目的との合致度
- 地域経済への貢献
- 社会的意義

**補助金ごとの政策目的例:**
- ものづくり補助金: 生産性向上、イノベーション
- 事業再構築補助金: 新分野展開、業態転換
- IT導入補助金: デジタル化、業務効率化
- 小規模事業者持続化補助金: 販路開拓、生産性向上

**良い申請書の特徴:**
- 政策目的への貢献が明確に記載
- 地域雇用創出や取引先への波及効果
- SDGs等の社会的視点

**減点される例:**
- 自社の利益のみを強調
- 政策目的への言及なし

---

### 4. 費用対効果（配点目安: 10-20%）

**評価ポイント:**
- 投資額と期待効果のバランス
- 積算根拠の妥当性
- 補助金に頼らない自走可能性

**良い申請書の特徴:**
- 設備投資の見積書が詳細
- 投資回収期間が明示されている
- 「補助金がなくても実施する」という本気度

**減点される例:**
- 積算が大雑把（「その他○○万円」が多い）
- 過剰な設備投資
- 補助金ありきの計画

## 文字数制限の一般的な目安

| セクション | 一般的な文字数制限 |
|---|---|
| 事業概要・要約 | 300-500字 |
| 事業の背景・目的 | 500-1000字 |
| 技術的内容・優位性 | 1500-2000字 |
| 事業化計画 | 1500-2000字 |
| 実施体制 | 500-1000字 |
| 売上計画・収支計画 | 1000-1500字 |

※補助金により異なるため、公募要領を必ず確認

## 必須添付資料（一般的なもの）

- [ ] 実施スケジュール（ガントチャート）
- [ ] 売上計画・収支計画表
- [ ] 組織図・実施体制図
- [ ] 設備投資の見積書
- [ ] 会社案内または事業概要
- [ ] 直近3期の決算書
- [ ] カタログ・図面（該当する場合）

---

## 審査員が好む構成

### 導入部（最初の1ページ）
1. 事業の核心を一言で
2. 現状の課題（数値で示す）
3. 解決策の概要
4. 期待される効果（数値で示す）

### 本文
- 見出しを活用（審査員が読みたい箇所にすぐ飛べる）
- 1段落 = 3-5行
- 重要な数値は太字または表組み
- 図表は必ず番号とキャプション

### まとめ
- 事業の意義を再強調
- 補助金の政策目的への貢献
- 実施への強い意欲

---

## 頻出する減点パターン

1. **抽象的な表現ばかり**
   - ×「画期的」「革新的」「世界初」
   - ○ 具体的な数値・比較データ

2. **主語が不明確**
   - ×「実施します」（誰が？）
   - ○「当社の開発部門（5名）が実施します」

3. **根拠のない楽観的予測**
   - ×「市場は拡大しているので売れる」
   - ○「○○調査によると市場は年率5%成長。当社の既存顧客20社へのヒアリングでも需要を確認」

4. **リスクへの言及なし**
   - 審査員は「この会社、リスクを理解しているのか？」と不安になる
   - 開発リスク、販売リスク、競合リスクと対策を明記

5. **図表の説明不足**
   - 図表だけ載せて本文で言及なし
   - 「図1のとおり、当社製品は従来比30%のコスト削減を実現」

---

## 審査員が評価する「数値」

| 項目 | 悪い例 | 良い例 |
|---|---|---|
| 市場規模 | 「市場は大きい」 | 「○○調査によると500億円市場」 |
| 売上目標 | 「5年後3億円」 | 「5年後3億円（既存顧客拡販1.5億円+新規開拓1.5億円）」 |
| 技術性能 | 「高速化を実現」 | 「処理時間を従来比50%短縮（実証試験データ添付）」 |
| 投資効果 | 「生産性向上」 | 「人時生産性20%向上、3年で投資回収」 |

## チェックリスト: 提出前の最終確認

### 内容面
- [ ] 全ての審査項目に回答している
- [ ] 数値に根拠がある
- [ ] 図表に番号とキャプションがある
- [ ] 専門用語に説明がある（審査員は必ずしも業界人ではない）

### 形式面
- [ ] 文字数制限を守っている
- [ ] 誤字脱字がない
- [ ] ページ番号がある（複数ページの場合）
- [ ] 必須添付書類が揃っている

### 印象面
- [ ] 最初の1ページで興味を引く内容か
- [ ] 図表が適切に使われているか
- [ ] 読みやすいレイアウトか
- [ ] 熱意が伝わるか

---

このリファレンスは、特定の補助金のテンプレート要件が不明な場合に使用します。
実際の審査基準は公募要領を必ず確認してください。
