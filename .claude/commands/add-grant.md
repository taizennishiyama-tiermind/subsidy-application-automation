# 新規補助金の追加

新しい補助金の種類をこの環境に追加します。

## 引数

$ARGUMENTS = 補助金名（省略時はユーザーに確認する）

## 手順

### Step 1: フォルダ構造の作成

以下のディレクトリとファイルを作成する:

```
grants/{補助金名}/
├── overview.md       ← 制度概要・申請要件・審査基準
├── templates/
│   └── README.md     ← テンプレートファイルの格納手順
└── scripts/
    └── fill_template.py  ← テンプレート書き込みスクリプト
```

### Step 2: overview.md の作成

ユーザーに以下の情報を聞き取るか、Webで調査する:
- 正式名称
- 所管省庁
- 公式サイトURL
- 制度概要（200字程度）
- 対象者要件
- 補助上限額・補助率
- 対象経費
- 申請書の主要記載項目
- 審査基準・配点
- 最新の公募スケジュール
- 採択のポイント
- よくある不採択理由

既存の `grants/*/overview.md` と同じ形式で作成する。

### Step 3: templates/README.md の作成

格納すべきテンプレートファイルの一覧とダウンロード元URLを記載する。

### Step 4: scripts/fill_template.py の作成

既存のスクリプト（例: `grants/ものづくり補助金/scripts/fill_template.py`）をベースに、
新しい補助金用のスクリプトを作成する。

- `EXCEL_CELL_MAPPING` と `WORD_PLACEHOLDER_MAPPING` はコメントアウトで用意
- テンプレートファイルがアップロードされた後に `/add-template` で実際のマッピングを設定

### Step 5: CLAUDE.md の更新

CLAUDE.md の対応補助金リストと grants 参照先に新しい補助金を追加する。

## 完了メッセージ

「{補助金名}を追加しました。次のステップ:
1. テンプレートファイル（Excel/Word）を `grants/{補助金名}/templates/` に配置してください
2. 配置後 `/add-template {補助金名}` でセルマッピングを登録してください」

## エラー時

- 同名のフォルダが既に存在する → 上書きするか確認
- Web調査で情報が見つからない → ユーザーに公募要領の提供を依頼
