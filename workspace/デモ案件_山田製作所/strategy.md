## 1. 選定補助金

### 第一候補：**ものづくり補助金（省力化・デジタル枠）**

| 項目 | 内容 |
|------|------|
| 正式名称 | ものづくり・商業・サービス生産性向上促進補助金 |
| 補助上限 | 4,500万円（省力化・デジタル枠） |
| 補助率 | 2/3（中小企業） |
| 申請窓口 | 全国中小企業団体中央会 |
| 採択率（直近） | 約48%（2024年第3次） |
| 公募期間 | 2026年4月〜6月（予定） |

#### 選定理由
1. **設備投資金額との適合**: 総投資額6,200万円のうち5,970万円が補助対象（MES・AI検査・プレス機）。補助上限4,500万円（2/3補助）に収まる
2. **省力化・デジタル化**: AI外観検査システム導入はデジタル枠の最重要加点要件に完全合致
3. **技術革新**: 精度±0.005mmへの向上（現行比2倍）は「革新的サービス・試作品開発」要件を満たす
4. **賃上げ要件**: 3%以上の賃上げ計画が加点要件と合致
5. **省エネ要件**: 省エネ型プレス機（旧型比40%削減）が環境加点に対応

#### 補助金額試算
- 補助対象経費: 5,970万円（設備購入費＋設置費）
- 補助率: 2/3
- 補助額: 3,980万円（上限4,500万円以内）
- 自己負担: 1,990万円（現預金4,100万円でカバー可能）

---

### 第二候補：**IT導入補助金（デジタル化基盤導入枠）**

| 項目 | 内容 |
|------|------|
| 補助上限 | 350万円 |
| 補助率 | 1/2〜3/4 |
| 対象 | MES（生産管理システム）部分のみ |

**活用方針**: ものづくり補助金との重複申請が可能な場合、MES（320万円）をIT導入補助金で対応し、ものづくり補助金の補助対象を設備費に集中させることで補助額を最大化する戦略も検討。

---

### 第三候補：**愛知県 産業立地・設備投資補助金（県単独補助）**

| 項目 | 内容 |
|------|------|
| 補助上限 | 1,000万円 |
| 補助率 | 10〜20% |
| 条件 | 愛知県内への設備投資、雇用維持・拡大 |

**活用方針**: ものづくり補助金採択後に上乗せ活用。国補助金との重複可否は愛知県産業振興課に確認要。

## 2. 申請戦略

### ストーリーパターン: **「課題解決型」**

```
現状（技術的優位性あるも設備老朽化・人手不足）
　↓
課題（品質検査の属人化・設備ダウン・受注機会損失）
　↓
解決策（AI検査×高速プレス機×MESで生産プラットフォーム刷新）
　↓
効果（生産性39%向上・不良率75%削減・年商2億3,000万円達成）
```

### 強調ポイント
1. **技術的革新性**: 精度±0.01mm→±0.005mm（業界最高水準）
2. **社会的意義**: 自動車EV化需要増への対応、製造業の人手不足解決モデル
3. **数値の確実性**: 受注確度の高い既存取引先からの増産依頼を根拠として提示
4. **賃上げコミットメント**: 補助金期間内に3%以上、中期的に継続賃上げを明約

### 加点要件対応状況

| 加点項目 | 対応 | 説明 |
|---------|------|------|
| 賃上げ計画（3%以上） | ✅ | 全従業員対象、2026年度内実施 |
| 省エネ設備 | ✅ | 旧型比40%省エネ実証値あり |
| デジタル化推進 | ✅ | MES＋AI検査で全工程デジタル化 |
| 炭素中立 | ✅ | CO2排出量22%削減試算 |
| 女性活躍 | ✅ | 女性管理職比率向上計画あり |

## 3. リスクと対応

| リスク | 確率 | 対応策 |
|--------|------|--------|
| 採択率の不確実性 | 中 | 第二候補（IT導入補助金）と並行申請の検討 |
| 設備納期遅延 | 低 | 採択後即発注、メーカーとの事前折衝 |
| 業者見積取得 | 低 | すでに2社から概算見積取得済み |
| 交付決定前の発注 | 高 | 補助金交付決定前の発注は補助対象外のためルール厳守 |
