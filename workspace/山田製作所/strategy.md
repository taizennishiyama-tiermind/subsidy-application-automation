## 第二候補: 中小企業省力化投資補助金（一般型）

### 想定効果

| 効果 | 内容 |
|------|------|
| 受け取れる補助金額 | 従業員数5〜20人では上限1,500万円、一定条件で2,000万円への引上げ余地あり |

| 申請書で訴えられる最大の強み | 段取り工数削減、熟練者依存の低減、若手でも再現可能な加工体制づくり |
| 採択時の事業効果 | 省力化による処理件数増加、納期短縮、人手不足対応 |

### なぜ第二候補か

今回の設備導入は、省力化と属人化解消の説明と非常に相性が良い。
一方で、山田製作所の訴求の核は「高付加価値化」にもあるため、設備投資の成果を付加価値向上で強く語れるものづくり補助金を第一候補に置いた。
逆に、人手不足対応や作業時間削減の定量効果を強く出せるなら、この制度が第一候補に逆転する余地はある。

## 第三候補: IT導入補助金（通常枠）

### 想定効果

| 効果 | 内容 |
|------|------|
| 受け取れる補助金額 | 1プロセス以上なら5万円以上150万円未満、4プロセス以上なら150万円以上450万円以下 |
| 御社のケースでの想定補助額 | CAMソフトや周辺ITのみ切り出せれば数十万〜数百万円規模 |
| 申請書で訴えられる最大の強み | 見積・製造準備・加工条件管理のデジタル化による業務効率化 |
| 採択時の事業効果 | 工程管理の標準化、入力ミス削減、準備時間短縮 |

### なぜ第三候補か

CAM更新や一部ソフトウェア導入には活用余地があるが、山田製作所の主投資は工作機械であり、案件全体との適合性は上位2制度に劣る。
設備本体を補助対象にしたい今回のケースでは、主制度というより部分最適の候補。

## 判断のポイント

- 今すぐ申請準備を進めたい場合: ものづくり補助金。既に公募が動いており、案件の主投資とも最も整合する
- 補助率を重視する場合: 中小企業省力化投資補助金（一般型）。省力化効果を定量化できれば有力
- 採択の筋道をわかりやすくしたい場合: ものづくり補助金。高付加価値化のストーリーを作りやすい
- ソフトウェア部分だけ切り出す場合: IT導入補助金。設備本体ではなくCAM更新中心の別案件としてなら検討余地がある

## 申請準備ガイド（ものづくり補助金）

- STEP D: 申請作業
  - テンプレート登録
  - 不足情報の補完
  - ドラフト作成
  - レビュー後に電子申請
