## ４．事業内容

### ①　今回の事業実施の背景（1000字以内）

当社は1985年の創業以来、愛知県において金属部品の切削加工を専業とする製造業者として40年の歴史を積み重ねてきた。主たる取引先は自動車部品メーカーであり、高精度な切削加工技術を武器に安定した受注関係を構築している。近年、自動車部品の軽量化・高精度化ニーズが高まる中で当社への引き合いは増加しており、受注機会は拡大している。

しかしながら、当社は現在、二つの構造的課題に直面している。

第一の課題は「熟練技術者の高齢化と技術継承の危機」である。当社の生産を支える主要な熟練工の多くは創業期から在籍するベテラン職人であり、今後3〜5年以内に退職を迎える見込みである。これらの職人が保有する高精度加工のノウハウ——切削条件の微調整、素材特性に応じた工具の使い分け、複雑形状への対応方法——は長年の経験から生まれた暗黙知であり、口頭伝達や見様見真似では容易に移転できない固有の技術資産である。現状の設備・体制では、この技術を次世代に継承する手段が存在しない。

第二の課題は「生産能力の限界による機会損失」である。受注増加が続く一方で、現行の汎用マシニングセンターでは複雑形状部品の加工に多くの段取り時間を要するため、稼働率が実質的に上限に達している。その結果、毎月数件の受注を断らざるを得ない状況が恒常化しており、売上機会の損失が続いている。

これら二つの課題を同時に解決し、持続的な事業成長を実現するために、今般の補助事業の実施を決断した。

### ②　会社全体の事業計画（1000字以内）

当社は今後5年間の経営目標として「技術の形式知化による生産革新と、自動車部品依存からの脱却」を掲げている。具体的には、5軸CNCマシニングセンターの導入を起点に、次の3段階の成長を描いている。

**第1段階（1〜2年目）：技術の形式知化と生産能力の回復**
DMG森精機製 DMU 50（5軸CNCマシニングセンター）を2台導入し、熟練工の加工ノウハウをNCプログラムとして蓄積する。具体的には、熟練工が担当してきた主要加工品目について、工具経路・切削条件・段取り手順を体系的にプログラム化し、当社独自の「加工ライブラリ」として管理する。この取り組みにより、熟練工が退職した後も同品質・同精度の加工を維持できる体制を構築する。

**第2段階（2〜3年目）：生産能力の拡大と機会損失の解消**
5軸加工機2台の本格稼働により、1台で複数面を同時加工できる5軸加工の利点を活かし、段取り時間を従来比で大幅に削減する。これにより現状の生産能力ボトルネックを解消し、これまで断ってきた受注を取り込む。3年後の売上目標は現状比15〜25%増（3.7〜4.0億円）とし、付加価値額を年率5〜8%増加させることを目指す。

**第3段階（3〜5年目）：航空・医療分野への参入**
5軸加工機で蓄積する高精度加工の実績と技術データを基盤として、航空部品（チタン・インコネル等の難削材）および医療機器部品（ステンレス・コバルトクロム合金等）という成長市場への参入を目指す。これらの分野は公差管理・品質保証の要求が極めて厳しく、当社が40年で培った精密加工の技術が直接的な差別化要因となる。

本補助事業はこの5カ年計画の起点であり、技術継承・生産革新・新分野開拓の三位一体を実現する不可欠な投資と位置づける。

（約700字）

### ③-1　今回の事業の具体的アクション（1000字以内）

本補助事業では、DMG森精機製 DMU 50（5軸マシニングセンター）2台の導入を実施する。以下のスケジュールで進める。

**設備導入スケジュール（全体18か月）**

| 月 | アクション |
|----|-----------|
| 1〜2か月目 | 機械レイアウト設計・工場床基礎工事・電気設備工事 |
| 3〜5か月目 | DMU 50（1号機）据付・試運転・検収 |
| 6〜8か月目 | 1号機を使用した熟練工によるNCプログラム作成・検証 |
| 9〜11か月目 | DMU 50（2号機）据付・試運転・検収 |
| 12〜14か月目 | 2号機稼働・加工ライブラリの整備・若手技術者への教育 |
| 15〜18か月目 | 両機本格量産稼働・加工データ収集・KPI測定開始 |

**NCプログラム化（形式知化）の進め方**

熟練工と若手技術者が協働して以下のプロセスを実施する。
1. 主要加工品目50種について、熟練工の加工動作を観察・記録
2. 使用工具・切削条件（回転数・送り速度・切込み量）・段取り手順をデジタルデータとして記録
3. DMU 50上でNCプログラムを試作・最適化
4. 試加工品の寸法測定・表面粗さ測定により品質を検証
5. 合格した加工プログラムを社内サーバーで一元管理

この仕組みにより、熟練工が退職した後も品質を維持できる「加工ライブラリ」を構築する。

**5軸加工機の技術的効果**

5軸同時制御加工により、従来の3軸加工機では複数回の段取りが必要だった複雑形状部品を、1回の段取りで完成させることができる。これにより段取り時間を従来比で50〜70%削減し、加工精度のばらつきも大幅に低減できる。

### ⑤　今回の事業の革新性・差別化（1000字以内）

本事業の革新性は、単なる設備の高度化にとどまらず、「熟練技術の形式知化」という製造業の本質的課題に取り組む点にある。

**革新性①：暗黙知のデジタル化による技術継承モデルの確立**

製造業における技術継承の問題は、団塊世代の大量退職が進む日本全体の社会課題でもある。多くの中小製造業が熟練工の退職に伴い技術を失い、品質低下や廃業に追い込まれている。当社は5軸CNCマシニングセンターを活用し、熟練工の「感覚・経験・勘」をNCプログラムというデジタルデータとして定量的に記録・再現する仕組みを構築する。これは愛知県の金属切削加工業において先進的な取り組みであり、地域の製造業モデルとして波及効果が期待できる。

**革新性②：5軸同時制御による生産方式の根本的転換**

現行の3軸マシニングセンターでは、複雑な形状の加工に際して複数回の段取り替えが必要であり、各段取りごとに位置決め誤差が蓄積するという技術的制約がある。DMU 50が持つ5軸同時制御機能を活用することで、1回のセッティングで複雑形状を完成させることができ、加工精度のばらつきを根本的に解消できる。これは自動車部品メーカーが求める公差±0.005mm以下の高精度要求にも対応できる水準である。

**差別化ポイント：航空・医療という成長分野への参入障壁の構築**

航空部品および医療機器部品の製造には、材料の特殊性（チタン・インコネル・コバルトクロム合金等の難削材）、厳格な品質管理体制（トレーサビリティ・工程管理）、高い加工精度が求められる。5軸加工の技術とNCプログラムライブラリを蓄積することで、これらの高参入障壁がある分野に参入できる技術基盤を構築する。一度参入すれば価格競争を避けた高付加価値受注が可能となり、既存の自動車部品加工との差別化が図れる。
