## 1. サービス概要

補助金申請書類の作成業務を AI で自動化・効率化する環境を構築し、納品するサービスです。

貴社の業務フローを対面でヒアリングした上で、最適な自動化環境を設計・構築します。構築した環境を使えば、**クライアント先へのヒアリングから申請書の作成・テンプレートへの書き込みまで**を AI アシスタントが一貫して支援します。

### 本サービスで実現できること

| 業務 | 従来 | 導入後 |
|------|------|--------|
| ヒアリングシートの作成 | 補助金ごとに手作業で作成 | 補助金の概要・申請書雛形を入力すると自動生成 |
| クライアントヒアリング | メモを取りながら手動整理 | 対話ログ・文字起こしデータから必要情報を自動抽出 |
| 財務データの分析 | 決算書を目視で読み取り | PDF/Excel を読み込ませて自動分析 |
| 最適な補助金の選定 | 経験と勘に依存 | 事業内容・財務状況から AI がマッチング提案 |
| 申請書ドラフト作成 | 一から手書き・転記 | 審査基準を踏まえた申請書を AI が自動生成 |
| テンプレートへの記入 | 手動で Excel/Word に転記 | 公式テンプレートへ自動書き込み |
| セルフレビュー | 経験者によるチェック | 審査基準に照らした品質チェックを AI が実施 |

## 2. サービスの進め方

### STEP 1: 対面ヒアリング・設計（実装に含む）

貴社事務所に伺い、以下を確認します。

- 現在の補助金申請業務フロー
- 使用しているテンプレート・書式
- よく申請する補助金の種類
- クライアントへのヒアリング方法
- 自動化によって解決したい課題

### STEP 2: AI 自動化環境の構築（実装 2時間）

ヒアリング内容を踏まえ、貴社専用の AI 環境を構築します。

**構築内容:**

- Claude Code 上で動作する補助金申請支援環境一式
- 対応補助金の設定（初期 4 種類、追加可能）
  - ものづくり補助金
  - IT導入補助金 / デジタル化・AI導入補助金
  - 新事業進出補助金（旧: 事業再構築補助金）
  - 小規模事業者持続化補助金
- 補助金ごとの制度概要・審査基準のナレッジベース
- ヒアリングシート自動生成機能
- 財務データ分析機能
- 申請書自動生成機能（ストーリーテリング対応）
- Excel/Word テンプレートへの自動書き込み機能
- 審査基準に基づくセルフレビュー機能
- スラッシュコマンドによる直感的な操作体系

**操作イメージ:**

```
/new-case       → 新規案件を開始し、AIがヒアリングを実施
/hearing        → ヒアリングを再開
/analyze        → 決算書を読み込み財務分析
/match          → 最適な補助金をAIが提案
/draft          → 採択率の高い申請書を自動生成
/review         → 審査基準でセルフレビュー
/fill           → 公式テンプレートに自動書き込み
```

### STEP 3: 納品物の引き渡し

以下一式を納品します。

| 納品物 | 内容 |
|--------|------|
| AI 自動化環境一式 | 上記構築内容のすべてのファイル・設定 |
| 運用マニュアル | セットアップ手順、操作方法、トラブルシューティング |
| カスタマイズガイド | 補助金の追加方法、テンプレートの更新方法 |

### STEP 4: レクチャー（1時間）

納品後、以下の内容で操作レクチャーを実施します。

- Claude Code の基本操作
- 実際の案件を使った一連のワークフロー体験
- カスタマイズ方法（補助金の追加、テンプレート更新等）
- 質疑応答

**完了基準:** 貴社ご担当者様が本ツールを使って一通り申請書を作成できることを確認した時点で、本案件をクローズとします。

## 8. 本サービスの特長

### なぜ AI 環境構築なのか

従来の「申請書作成代行」とは異なり、**貴社が自走できる環境**を構築・納品します。

- 一度構築すれば、何度でも繰り返し使える
- 新しい補助金にも自分で対応できる（`/add-grant` コマンド）
- テンプレートが更新されても自分で再設定できる（`/add-template` コマンド）
- AI がプロの知見で逆質問するため、クライアントへのヒアリング品質が安定する
- 属人化を防ぎ、担当者が変わっても同じ品質で申請書を作成できる

### 対応補助金の拡張性

初期設定の 4 補助金に限らず、`/add-grant` コマンドで**いつでも新しい補助金を追加**できます。補助金の概要情報とテンプレートを用意するだけで、同じワークフローで申請書作成が可能になります。
