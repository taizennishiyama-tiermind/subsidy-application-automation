# ものづくり補助金 参考資料フォルダ

公募要領・交付規定・Q&Aなどの公式資料を格納してください。
格納されたファイルは `/match`・`/draft`・`/review` 実行時に AI が自動で参照します。

---

## 格納する資料の種類

| 資料の種類 | ファイル名の例 | 用途 |
|---|---|---|
| 公募要領 | `公募要領_第22次.pdf` | 申請要件・審査基準・スケジュールの根拠 |
| 交付規定 | `交付規定.pdf` | 対象経費・禁止事項の詳細確認 |
| Q&A・FAQ | `QA_第22次.pdf` | 申請上の疑問点の整理 |
| 審査基準の詳細 | `審査基準説明.pdf` | 審査ポイントの深掘り |
| パンフレット・概要資料 | `パンフレット2026.pdf` | 制度の概要把握 |
| その他参考資料 | 任意 | 採択傾向の分析等 |

## 注意事項

- 公募回ごとに要件・様式が変わることがあります。最新回の資料を格納してください
- 古い回の資料は `docs/archive/` サブフォルダに移動して保管することを推奨します
- PDF はテキスト抽出可能なものを使用してください（スキャンPDFは内容を読み取れません）
