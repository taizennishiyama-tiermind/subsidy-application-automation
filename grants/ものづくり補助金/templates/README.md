# ものづくり補助金 テンプレートファイル

## 格納するファイル

以下のファイルをこのフォルダに配置してください:

1. **事業計画書テンプレート** (.xlsx or .docx)
   - 公募要領からダウンロードした公式テンプレート
   - ファイル名例: `事業計画書_テンプレート.xlsx`

2. **経費明細テンプレート** (.xlsx)
   - 補助対象経費の内訳を記入するExcelファイル
   - ファイル名例: `経費明細_テンプレート.xlsx`

3. **その他様式** (任意)
   - 賃金引上げ計画表
   - 事業実施体制表
   - 等

## ダウンロード元

公式サイト: https://portal.monodukuri-hojo.jp/about.html

※ 公募回ごとにテンプレートが変わる場合があります。必ず最新の公募要領に対応したテンプレートを使用してください。

## テンプレート追加後の手順

1. テンプレートファイルをこのフォルダに配置
2. `/add-template ものづくり補助金` を実行
3. AIがテンプレートの構造を解析し、スクリプトのセルマッピングを更新
