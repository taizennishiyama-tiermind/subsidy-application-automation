# IT導入補助金 / デジタル化・AI導入補助金 テンプレートファイル

## 格納するファイル

以下のファイルをこのフォルダに配置してください:

1. **事業計画書テンプレート** (.xlsx or .docx)
   - ファイル名例: `IT導入_事業計画書_テンプレート.xlsx`

2. **導入ITツール情報シート** (.xlsx)
   - 導入するITツールの詳細情報
   - ファイル名例: `ITツール情報_テンプレート.xlsx`

3. **その他様式** (任意)
   - 労働生産性計算シート
   - セキュリティ対策チェックリスト

## ダウンロード元

公式サイト: https://it-shien.smrj.go.jp/

※ IT導入補助金は電子申請（jGrants）がメインですが、事前準備用のテンプレートを格納してください。

## テンプレート追加後の手順

1. テンプレートファイルをこのフォルダに配置
2. `/add-template IT導入補助金` を実行
3. AIがテンプレートの構造を解析し、スクリプトのセルマッピングを更新
