# IT導入補助金 / デジタル化・AI導入補助金

## 制度概要

中小企業・小規模事業者等が自社の課題やニーズに合ったITツールを導入する経費の一部を補助し、業務効率化・売上アップを支援する制度。

- 2025年度名称: IT導入補助金2025
- 2026年度名称: **デジタル化・AI導入補助金2026**（名称変更）
- 所管: 中小企業庁 / 独立行政法人中小企業基盤整備機構
- 公式サイト: https://it-shien.smrj.go.jp/
- 変更点: 2026年度からAI活用・業務変革をより重視する制度設計に移行

## 申請枠（2026年度）

### 1. 通常枠
- 対象: ソフトウェア、クラウドサービス等のITツール導入
- 補助額: 5万円〜450万円以下
- 補助率: 1/2以内

### 2. インボイス枠
- 対象: インボイス対応のITツール導入
- 補助額: 〜350万円
- 補助率: 2/3〜3/4（条件による）

### 3. セキュリティ対策推進枠
- 対象: サイバーセキュリティ対策
- 補助額: 5万円〜100万円
- 補助率: 1/2以内

### 4. 複数社連携IT導入枠
- 対象: 複数の中小企業が連携してITツールを導入
- 補助額: 〜3,000万円
- 補助率: 1/2〜2/3

## 申請要件

1. 中小企業・小規模事業者等であること
2. 日本国内で法人登記され事業を営んでいること
3. gBizIDプライムのアカウントを取得していること
4. 独立行政法人情報処理推進機構（IPA）が実施する「SECURITY ACTION」の宣言を行うこと
5. 「みらデジ経営チェック」の実施

## IT導入支援事業者（ITベンダー）の関与

- 申請は必ず「IT導入支援事業者」と共同で行う
- ITツールは事前に事務局に登録されたものから選択
- 支援事業者がツールの導入・サポートを実施

## 申請書の主要記載項目

### 事業計画
- 自社の経営課題の認識
- 経営改善に向けた具体的な問題意識
- 導入するITツールの選定理由
- 導入効果の見込み

### 数値目標
- 労働生産性の向上率
- 売上高・経常利益の改善見込み
- 生産性向上の具体的な数値

## 審査基準

### 事業面
- 自社の経営課題を理解しているか
- 経営改善に向けた具体的な問題意識があるか
- 課題分析・将来計画に対し、ITツール導入が適切にマッチしているか

### 計画目標値の審査
- 労働生産性の伸び率が妥当か
- 導入効果が事業計画と整合しているか

### 政策面
- 生産性向上への貢献
- 地域経済への波及効果

### 加点項目
- くるみん・えるぼし認定
- 地域DX促進活動の取組
- インボイス制度への対応
- サイバーセキュリティお助け隊の活用

## 採択のポイント

1. **課題と解決策の一貫性**: 経営課題 → ITツール → 改善効果の論理的つながり
2. **具体的な数値目標**: 「○○時間削減」「○○%効率化」等
3. **ITツールの適合性**: なぜそのツールが最適なのか説明
4. **SECURITY ACTION宣言**: 事前に忘れずに実施
5. **みらデジ経営チェック**: 事前に実施必須
6. **IT導入支援事業者との連携**: 信頼できる事業者を選定

## よくある不採択理由

- 経営課題とITツールの関連が不明確
- 数値目標の根拠が弱い
- 導入効果の説明が抽象的
- SECURITY ACTION未宣言
- みらデジ経営チェック未実施
