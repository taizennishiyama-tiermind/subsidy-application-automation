# IT導入補助金 参考資料フォルダ

公募要領・交付規定・Q&Aなどの公式資料を格納してください。
格納されたファイルは `/match`・`/draft`・`/review` 実行時に AI が自動で参照します。

---

## 格納する資料の種類

| 資料の種類 | ファイル名の例 | 用途 |
|---|---|---|
| 公募要領 | `公募要領_2026年度.pdf` | 申請要件・審査基準・スケジュールの根拠 |
| 交付規定 | `交付規定.pdf` | 対象経費・禁止事項の詳細確認 |
| Q&A・FAQ | `QA_2026.pdf` | 申請上の疑問点の整理 |
| IT導入支援事業者向け説明資料 | `ベンダー説明資料.pdf` | IT導入支援事業者との連携に必要な情報 |
| 対象ツール・サービス一覧 | 任意 | 申請可能なITツールの確認 |
| パンフレット・概要資料 | `パンフレット2026.pdf` | 制度の概要把握 |

## 注意事項

- 2026年度より「デジタル化・AI導入補助金」に名称変更予定。最新の名称・要件をご確認ください
- PDF はテキスト抽出可能なものを使用してください（スキャンPDFは内容を読み取れません）
- 古い年度の資料は `docs/archive/` サブフォルダに移動して保管することを推奨します
