# 小規模事業者持続化補助金 参考資料フォルダ

公募要領・交付規定・Q&Aなどの公式資料を格納してください。
格納されたファイルは `/match`・`/draft`・`/review` 実行時に AI が自動で参照します。

---

## 格納する資料の種類

| 資料の種類 | ファイル名の例 | 用途 |
|---|---|---|
| 公募要領（商工会議所版） | `公募要領_第19回_商工会議所.pdf` | 申請要件・審査基準・スケジュールの根拠 |
| 公募要領（商工会版） | `公募要領_第19回_商工会.pdf` | 商工会経由の申請に必要 |
| 交付規定 | `交付規定.pdf` | 対象経費・禁止事項の詳細確認 |
| Q&A・FAQ | `QA_第19回.pdf` | 申請上の疑問点の整理 |
| 経営計画書・補助事業計画書の記載例 | `記載例.pdf` | 作成の参考 |
| 審査基準の詳細 | `審査基準説明.pdf` | 審査ポイントの深掘り |

---

## ダウンロード元

- 商工会議所版: https://r3.jizokukahojokin.info/
- 商工会版: https://www.shokokai.or.jp/jizokuka_r1h/

---

## 注意事項

- 商工会議所版と商工会版で申請窓口・様式が異なります。クライアントの管轄を確認してください
- 公募回ごとに要件・申請締切が変わります。最新回の資料を格納してください
- PDF はテキスト抽出可能なものを使用してください（スキャンPDFは内容を読み取れません）
- 古い公募回の資料は `docs/archive/` サブフォルダに移動して保管することを推奨します
