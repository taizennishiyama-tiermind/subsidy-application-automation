# 小規模事業者持続化補助金 テンプレートファイル

## 格納するファイル

以下のファイルをこのフォルダに配置してください:

1. **様式2: 経営計画書兼補助事業計画書** (.docx or .xlsx)
   - ファイル名例: `様式2_経営計画書.docx`

2. **様式3: 補助事業計画書** (.docx or .xlsx)
   - ファイル名例: `様式3_補助事業計画書.docx`

3. **経費明細表** (.xlsx)
   - ファイル名例: `経費明細表.xlsx`

4. **その他様式** (任意)
   - 事業支援計画書（様式4）は商工会議所/商工会が作成

## ダウンロード元

- 商工会議所地区: https://r6.jizokukahojokin.info/
- 商工会地区: https://www.jizokukanb.com/

## テンプレート追加後の手順

1. テンプレートファイルをこのフォルダに配置
2. `/add-template 小規模事業者持続化補助金` を実行
3. AIがテンプレートの構造を解析し、スクリプトのセルマッピングを更新
