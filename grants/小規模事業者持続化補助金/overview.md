# 小規模事業者持続化補助金

## 制度概要

小規模事業者が経営計画を策定して取り組む販路開拓や業務効率化の取組を支援する制度。物価高騰や賃金引上げに対応する事業者を重点的に支援。

- 正式名称: 小規模事業者持続化補助金
- 所管: 中小企業庁
- 実施機関: 日本商工会議所（商工会議所地区）/ 全国商工会連合会（商工会地区）
- 公式サイト（商工会議所地区）: https://r6.jizokukahojokin.info/
- 公式サイト（商工会地区）: https://www.jizokukanb.com/

## 支援類型（2026年度）

1. **一般型** - 個社での販路開拓支援
2. **創業型** - 創業間もない事業者向け
3. **共同・協業型** - 複数事業者の連携による取組
4. **ビジネスコミュニティ型** - 地域の事業者グループ向け

## 対象者の要件

以下の5つを全て満たすこと:

1. **小規模事業者であること**
   - 商業・サービス業（宿泊・娯楽除く）: 従業員5人以下
   - サービス業のうち宿泊・娯楽業: 従業員20人以下
   - 製造業その他: 従業員20人以下
2. 資本金5億円以上の法人に100%保有されていないこと
3. 課税所得の年平均額が15億円を超えていないこと
4. 商工会議所または商工会の管轄地域内で事業を営んでいること
5. 過去に採択された場合は報告書が受領済みであること

## 申請書の構成

### 経営計画書
1. **企業概要**
   - 会社名、代表者名、住所
   - 業種、設立年月
   - 従業員数、売上高

2. **顧客ニーズと市場の動向**
   - ターゲット顧客の特性
   - 市場規模・動向
   - 競合状況

3. **自社や自社の提供する商品・サービスの強み**
   - 他社との差別化ポイント
   - 独自の技術・ノウハウ
   - 顧客からの評価

4. **経営方針・目標と今後のプラン**
   - 中長期的な経営方針
   - 具体的な目標数値
   - 実行計画

### 補助事業計画書
1. **補助事業で行う事業名**
2. **販路開拓等の取組内容**
3. **業務効率化（生産性向上）の取組内容**
4. **補助事業の効果**

## 審査基準

### 基礎審査
- 必要書類が全て揃っているか
- 要件に適合しているか

### 計画審査
- 自社の経営状況分析の妥当性
- 経営方針・目標と補助事業計画の一貫性
- 補助事業計画の有効性
- 積算の透明・適切性

### 加点審査
- 賃金引上げの取組
- 経営力向上計画の認定
- 事業承継の取組
- 過去の受賞歴（経営革新計画等）
- 地域資源活用

## 採択のポイント

1. **わかりやすい文章**: 審査員は多くの申請書を読む。簡潔明瞭に
2. **自社の強みを明確に**: 「なぜ自社がこの事業をやるべきか」を説得力をもって
3. **市場分析**: 地域の市場動向、顧客ニーズを具体的に
4. **数値目標**: 売上・利益の改善見込みを具体的な数字で
5. **商工会議所/商工会の支援**: 事業支援計画書の発行が必要（締切に注意）
6. **写真・図表の活用**: 店舗・商品の写真、市場データのグラフ

## 商工会議所/商工会との連携

- 申請には「事業支援計画書」の発行が必須
- 発行受付の締切は申請締切の約2週間前
- 早めに相談し、アドバイスを受けることが重要
- 商工会議所の経営指導員による支援を活用

## よくある不採択理由

- 経営計画と補助事業の関連が弱い
- 市場分析が不十分
- 自社の強みが不明確
- 補助事業の効果が具体的でない
- 事業支援計画書の発行が間に合わない
- 経費の積算根拠が不明確

## 採択率の傾向

- 平均的な採択率: 約50〜65%（回によって変動）
- 他の補助金と比較して申請のハードルは低め
- ただし通常枠は補助額が小さいため、投資対効果を考慮
