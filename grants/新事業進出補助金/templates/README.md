# 新事業進出補助金（旧: 事業再構築補助金）テンプレートファイル

## 格納するファイル

以下のファイルをこのフォルダに配置してください:

1. **事業計画書テンプレート** (.xlsx or .docx)
   - ファイル名例: `新事業進出_事業計画書_テンプレート.xlsx`

2. **収益計画テンプレート** (.xlsx)
   - 付加価値額・収益計画の計算シート
   - ファイル名例: `収益計画_テンプレート.xlsx`

3. **認定支援機関確認書** (.docx)
   - 認定経営革新等支援機関の確認書テンプレート

4. **その他様式** (任意)
   - 経費明細表
   - 実施体制表

## ダウンロード元

公式サイト: https://shinjigyou-shinshutsu.smrj.go.jp/

## テンプレート追加後の手順

1. テンプレートファイルをこのフォルダに配置
2. `/add-template 新事業進出補助金` を実行
3. AIがテンプレートの構造を解析し、スクリプトのセルマッピングを更新
