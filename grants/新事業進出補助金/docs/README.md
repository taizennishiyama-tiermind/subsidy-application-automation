# 新事業進出補助金 参考資料フォルダ

公募要領・交付規定・Q&Aなどの公式資料を格納してください。
格納されたファイルは `/match`・`/draft`・`/review` 実行時に AI が自動で参照します。

---

## 格納する資料の種類

| 資料の種類 | ファイル名の例 | 用途 |
|---|---|---|
| 公募要領 | `公募要領_第3回.pdf` | 申請要件・審査基準・スケジュールの根拠 |
| 交付規定 | `交付規定.pdf` | 対象経費・禁止事項の詳細確認 |
| Q&A・FAQ | `QA_第3回.pdf` | 申請上の疑問点の整理 |
| 事業計画書作成の手引き | `事業計画書手引き.pdf` | 記載要領・注意事項の確認 |
| 審査基準の詳細 | `審査基準説明.pdf` | 審査ポイントの深掘り |
| パンフレット・概要資料 | `パンフレット.pdf` | 制度の概要把握 |

## 注意事項

- 旧・事業再構築補助金から新事業進出補助金に制度が変更されています（2025年4月〜）
- 公募回ごとに対象範囲・要件が変わることがあります。最新回の資料を必ず格納してください
- PDF はテキスト抽出可能なものを使用してください（スキャンPDFは内容を読み取れません）
- 古い公募回の資料は `docs/archive/` サブフォルダに移動して保管することを推奨します
